Pymaze JS

This is a live demo of Pymaze JS. Please select the edition you wish to try:

  • Modern (tested on Firefox 38 and newer)
  • Legacy (tested on Chrome 43 and newer, and Internet Explorer 11 and newer)

If you don't have Firefox 38 or newer, please use the Legacy edition. Despite the name, the Legacy edition has exactly the same features as the Modern edition. It may however be slower and more difficult to debug.

Authors and Contributors

Pymaze was the group project of SJAM's CS Club in 2015. Fengyang Wang (@TotalVerb) ported the original Pymaze, written in Python, to Javascript.
